# Roadmap

## MVP

- [ ] User auth (marketer vs. influencer role)
- [ ] Influencer profiles (niche, platforms, audience size, bio)
- [ ] Opportunity listings (posted by marketers)
- [ ] Browse/search opportunities (for influencers)
- [ ] Browse/search influencers (for marketers)
- [ ] Apply to / express interest in an opportunity
- [ ] Basic messaging or contact flow

## Post-MVP

- [ ] Saved searches / alerts
- [ ] Match scoring / recommendations
- [ ] Campaign tracking
- [ ] Reviews and ratings
- [ ] Analytics dashboard for marketers
