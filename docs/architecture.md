# Architecture

> This document will evolve as decisions are made. Start here when onboarding.

## Overview

HAMO is a two-sided marketplace for B2B SaaS influencer marketing. The two user types are:

- **Marketers** — teams at B2B SaaS companies looking to run influencer campaigns
- **Influencers** — creators, thought leaders, and subject matter experts with relevant audiences

## Core Concepts

| Concept | Description |
|---|---|
| **Opportunity** | A campaign or collaboration posted by a marketer |
| **Profile** | An influencer's public presence — niche, audience size, platforms |
| **Match** | A connection made between an opportunity and an influencer |
| **Application** | An influencer expressing interest in an opportunity |
